# Text RPG Game

A simple text-based RPG game written in Python as a practice project for learning object-oriented programming.

The player fights random enemies, earns gold and XP, buys better weapons, uses health potions, and levels up by defeating enemies.

## Features

- Turn-based combat
- Random enemies (Goblin, Skeleton, and Orc)
- Weapon shop
- Health potion system
- Inventory
- XP and leveling system
- Gold rewards
- Player and enemy status
- Game over statistics
- Automatic health restoration when leveling up

## Built With

- Python 3
- PyCharm
